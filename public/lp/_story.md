# 君斗りんくのすれ違ひ通信 ― サウンドノベル脚本

主人公「私」の一人称・旅日記。スクロールで一文ずつ送られる。
かまいたちの夜／弟切草の質感：暗い情景、間（ま）、ほのかな不穏と神秘、誰かの先行する足跡。
ホラーにはしない。叙情とノスタルジー。旧仮名遣い混じり。

季節：冬(吹雪)→春(桜)→夏昼(蝉・川)→夏夜(祭り)→秋(紅葉)→晴(結)
各文 20〜45字。合計約80文。30分かけて読める分量。

---

## 序章 — 扉（winter / 闇に浮かぶ語り）
id: prologue
- ある冬の夜のことを、まだ覚えている。
- ひとり、雪の峠を越えようとしていた。
- なぜ歩いていたのか、いまはもう、思い出せない。
- ただ、行かねばならぬ気が、していた。
- 月日は百代の過客にして、行き交ふ年もまた旅人なり。
- ――旅は、もう、はじまっていた。

## 一章 — 雪国へ（winter / 汽車・トンネル・雪国）
id: yukiguni
- 汽笛が、ポオォと尾を引いた。
- 列車は、長いトンネルへ吸い込まれてゆく。
- ガタン、ゴトン。闇のなかを、ただ運ばれる。
- ――トンネルを抜けると、そこは雪国であった。
- 窓の外は、見わたすかぎりの白。
- 雪は、しんしんと降りつもる。
- 温泉に、猿が一匹、目を閉じていた。
- 私は、この景色を、誰かに見せたかった気がする。
- けれど、隣の席は、もう空いていた。

## 二章 — 雪道の足跡（winter / 蓑笠の旅人・先行する足跡）
id: yukimichi
- 駅を出て、雪の一本道を歩く。
- ザッ、ザッ、と、雪を踏みしめる音だけが響く。
- ふと、前を見ると――足跡が、続いていた。
- 私のものではない。誰かが、先に通ったのだ。
- その人も、ひとりだったのだろうか。
- 足跡は、消えずに、白い道に刻まれていた。
- 私も、いま、足跡を残している。
- いつか誰かが、これをたどるのかもしれない。

## 三章 — 春・桜（spring / 雪解け・桜・人力車）
id: haru
- 峠を下るほどに、雪は、霞へと変わった。
- 気づけば、桜が、ひとひら、頬をかすめる。
- からから、と、人力車が過ぎてゆく。
- ただ通り過ぎただけの、なんでもない辻。
- けれど、その辻を、私はあとで思い出すことになる。
- 「いづれの場所も、旅の途中なり」
- 道ばたの地蔵に、誰かが供えた花が、まだ新しかった。
- 私より先に、ここで手を合わせた人がいる。
- すれ違うとは、こういうことかもしれない。

## 四章 — 一服の間（spring / 茶室・休息・庭の鹿威し）
id: chaseki
- 街道の茶屋で、ひと息つくことにした。
- 一服の茶が、冷えた指に、しみわたる。
- 庭の隅で、鹿威しが、水を溜めている。
- ……。
- コトン。
- 竹が石を打つ、その音だけが、時を刻む。
- 静けさが、かえって、深くなる。
- 一期一会、と、誰かが言った。
- この一杯も、この一刻も、二度とは巡らない。

## 五章 — 夏・昼（summer-day / 蝉・川・かき氷・儚さ）
id: natsu-hiru
- 季節は、いつのまにか、夏になっていた。
- 蝉時雨が、降るように、満ちている。
- 川がさらさらと流れ、魚が、ひとつ跳ねた。
- 風鈴が、ちりりと鳴る。
- 幹には、抜け殻がひとつ。もう、鳴かない。
- かき氷は、紅い雫を残して、溶けてゆく。
- 夏は、いつか終わる。
- だからこそ、歩いた今日が、あとで愛おしくなる。
- 麦わら帽子が、ふわりと、風にさらわれた。

## 六章 — 夏・夜祭り（summer-night / 屋台・花火・すれ違い・線香花火）
id: natsu-yoru
- 日が暮れて、遠くに、祭りの灯が見えた。
- たこ焼き、焼きそば、提灯の列。
- 人ごみのなか、誰かと肩がふれた。
- 振り向いたが、もう、その背中は人波に消えていた。
- ドオン――と、花火が、夜空をひらく。
- 同じ花火を、見上げている誰かが、いるはずだ。
- 言葉は、交わさなくていい。
- ただ、同じ道を、同じ夜を、踏んだという、それだけで。
- 線香花火が、ぽとりと、落ちた。
- すれ違いとは、さみしさの、もうひとつの名前だ。

## 七章 — 時を越えて（summer-night→autumn / 過去30日・タイムシフト・推しの軌跡）
id: toki
- 祭りのあと、私は、ひとつのことに気づいた。
- 同じ刻に、居合わせなくても、いいのだと。
- 過ぎた三十日の道のりが、人と人を、そっと結ぶ。
- 私が今日たどった道を、昨日、誰かが歩いていた。
- その人の足あとを、私は、知らずに踏んでいた。
- 慕わしいあの人の通った道さえ、たどれるという。
- 会えずとも、同じ景色を、確かに、見た。
- それは、時を越えた、すれ違い。

## 八章 — 秋・残るもの（autumn / 紅葉・落ち葉・足あとは消えない）
id: aki
- 木々が、燃えるように、色づいていた。
- 落ち葉を踏む音が、かさり、と鳴る。
- どこからか、焼き芋の、匂い。
- 消えてしまっては、思い出には、ならない。
- だから私は、歩いた道を、消さずにおく。
- 幾年たっても、あの日の辻へ、帰れるように。
- ふり返れば、足あとは、ずっと続いていた。
- 冬の雪道の、あの足跡まで、ひと続きに。
- 旅とは、消えない一本の線を、引くことだった。

## 結章 — 扉（clear / 富士・月・流れ星・X誘い）
id: epilogue
- 雲が晴れ、はじめて、富士が見えた。
- 湖に、月が、逆さに浮かんでいる。
- すうっと、流れ星が、ひとつ。
- 願いを言う間もなく、消えていった。
- けれど――願いは、もう、足が叶えている。
- 想いが強ければ、足は、自ずと動いているのだから。
- 同じ道は、二度とない。だから、足あとは消えない。
- あなたの旅も、ここから、はじまる。
- ――さあ、消えてしまう前に。
